# HealthPulse - Wellness Optimization Platform

HealthPulse is a professional health-tech analytics platform dedicated to **SDG 3: Good Health and Well-being**. It empowers users to track their physical and mental health markers through structured routines, personalized challenges, and real-time resilience coaching.

## 🚀 Key Features

- **Personalized Onboarding**: Intelligent identity selection (Guest/Member) and goal-setting infrastructure.
- **Health Resilience Index**: A unified metric calculating wellness based on habits, challenges, and psychometric stability.
- **Dynamic Challenges**: Goal-dependent wellness protocols that refresh daily.
- **Routine Management**: Clinical habit tracking with immediate feedback.
- **Cognitive Monitoring**: Psychometric state tracking with data-driven feedback.
- **Clinical Dashboard**: Integrated analytics and priority recommendations for systemic optimization.

## 🛠️ Technical Overview

- **Core**: HTML5, Vanilla JavaScript.
- **Styling**: Modern CSS3 with custom design tokens, mesh gradients, and glassmorphism.
- **Persistence**: Localized data storage for clinical privacy.
- **Animations**: CSS animations and `Canvas-Confetti` for behavioral reinforcement.

## 📂 Project Structure

- `/index.html`: Entry onboarding experience.
- `/home.html`: Personalized briefing and health assessment.
- `/dashboard.html`: Centralized analytics hub.
- `/habits.html`: Routine protocol verification.
- `/challenges.html`: Resilience challenge execution.
- `/mood.html`: Mental health monitoring.
- `/profile.html`: Identity and goal management.
- `/js/`: System logic and shared protocols.
- `/css/`: Design system and component styling.
